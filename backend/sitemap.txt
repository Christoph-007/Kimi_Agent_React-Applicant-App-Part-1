## 🔗 Navigation Flow Diagram (Text)

```
[ Landing / ] ──→ [ Login /login ] ──→ Role-based redirect
      │                                        │
      ├──→ [ /signup/applicant ]               ├──→ /applicant/dashboard (Applicant)
      ├──→ [ /signup/employer ]                ├──→ /employer/dashboard  (Employer)
      └──→ [ /jobs ] ──→ [ /jobs/:id ]         └──→ /admin/dashboard     (Admin)

APPLICANT PORTAL (/applicant/*)                    ← POST /api/auth/logout on Navbar
  dashboard → jobs → applications → shifts → attendance
           → job-requests → notifications → profile → resume → settings

EMPLOYER PORTAL (/employer/*)                      ← POST /api/auth/logout on Navbar
  dashboard → jobs → jobs/create
           → jobs/:id/edit → jobs/:id/applicants
           → applicants → applicants/:id
           → shortlist → job-requests
           → shifts → shifts/create → attendance
           → notifications → profile

ADMIN PORTAL (/admin/*)                            ← POST /api/auth/logout on Navbar
  dashboard → employers → applicants → jobs → analytics
```

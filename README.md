# ShiftMatch

A full-stack job matching platform for shift workers.

## Project Structure

- app/ — Applicant React frontend (port 5173)
- employer/ — Employer React frontend (port 5174)
- admin/ — Admin React frontend (port 5175)
- backend/ — Node.js + Express + MongoDB API (port 5005)

## Getting Started

### Backend
cd backend
npm install
npm run dev

### Applicant App
cd app
npm install
npm run dev

### Employer App
cd employer
npm install
npm run dev

### Admin App
cd admin
npm install
npm run dev

## Tech Stack
- Frontend: React + TypeScript + Vite + Tailwind + shadcn/ui
- Backend: Node.js + Express + MongoDB + Mongoose
- Auth: JWT
- File Storage: Cloudinary
- Email: Nodemailer
